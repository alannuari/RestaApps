-   API: [RestaurantAPI]()

### Cara Menjalankan Project

-   Buka code editor di dalam folder project
-   Jalankan terminal di dalam code editor
-   Install semua package

    > `npm install`

-   Untuk tahap development

    > `npm run start-dev`

-   Untuk tahap production
    > `npm run build`
